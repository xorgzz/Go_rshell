but seriously anything you do with this script is only on you.

### How to set up
Pull git repo <br>
`$ git clone https://github.com/xorgzz/go_rshell` <br>
Build script <br>
`$ cd go_rshell` <br>
`$ go build`<br>
Run exec <br>
`$ ./go_rshell`

### Specifics
 * Default ip is 127.0.0.1 
 * Default port is 2000
